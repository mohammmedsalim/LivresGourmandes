# Livres Gourmands - Projet Web Avancé (Étape 3)

## Contexte du Projet
**LivresGourmands.net** est une plateforme e-commerce spécialisée dans la vente de livres de cuisine. Ce projet académique vise à mettre en œuvre une architecture complète (Frontend React + Backend Node.js) avec une attention particulière portée à l'expérience utilisateur (UX/UI) et à la qualité du code.

## Étape 3 : Conception Visuelle et Intégration

Cette étape se concentre sur :
1.  La définition de l'identité visuelle et l'expérience utilisateur.
2.  La production des maquettes et prototypes.
3.  L'initialisation de la structure technique du projet.

## Contenu du Dépôt

*   `/docs` : Documentation complète (Architecture, Conception Visuelle).
*   `/frontend` : Code source de l'application React (à venir).
*   `/backend` : Code source de l'API Express (à venir).

## Livrables de l'Étape 3

### 1. Conception
*   **Charte Graphique** : Couleurs, typographies et composants UI définis.
*   **Wireframes** : Structure des pages clés (Accueil, Produit, Panier).
*   **Prototype** : Scénarios utilisateurs interactifs.

### 2. Technique
*   **Architecture** : Découpage Client/Serveur, choix des technologies (MERN Stack).
*   **Base de Données** : Schéma relationnel MySQL prévu.
*   **Panier** : Logique de gestion via Context API et LocalStorage.

1.  **Backend** : `cd backend && npm install && npm run dev`
2.  **Frontend** : `cd frontend && npm install && npm run dev`
3.  **DB** : Importer `backend/database.sql`
